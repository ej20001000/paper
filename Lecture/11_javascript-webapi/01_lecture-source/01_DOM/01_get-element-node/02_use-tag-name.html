02. 태그 이름을 이용한 요소 노드 취득

        
            Document.prototype/Element.prototype.getElementsByTagName 메소드는
            인자로 전달한 태그 이름을 갖는 모든 요소 노드를 탐색하여 반환한다.
            메소드 이름의 Elements가 복수형인 것에서 알 수 있듯 여러 개의 요소 노드 객체를 갖는
            DOM 컬렉션 객체인 HTMLCollection 객체를 반환한다
